# Deployment & Context

I dag skal vi se nærmere på deling af state på tværs af hele vores React-app.

Derudover skal vi prøve at deploy vores applikation, der findes mange løsningsmuligheder til dette, men vi kommer til at bruge Github Pages. Dette spiller nemt sammen med vores Github repository, og så er det både gratis og giver en url med et hosted domæne vi kan sende til UX.

## Læringsmål

1. Context
   1. Forstå problematikken bag ved state der skal deles på tværs af hele applikationen og hvordan vi undgår "Prop drilling" igennem mange/alle komponenter.
2. Deployment
   1. Forstå konceptet omkring 'deployment', og nogle af de steps der skal til for at komme fra `localhost`, til et 'deployed site', som brugere kan tilgå alle steder.

## Øvelser

1. Gemme login oplysninger i Routing Context.
2. Deploy jeres `booking-site` til GitHub Pages.
